## 🚀 快速开始

### 1. 安装依赖

\`\`\`bash
npm install
\`\`\`

### 2. 配置 API Key (可选)

复制 `.env.example` 为 `.env`:

\`\`\`bash
cp .env.example .env
\`\`\`

编辑 `.env` 文件，填入你的 Gemini API Key:

\`\`\`env
VITE_GEMINI_API_KEY=your_actual_api_key_here
\`\`\`

**获取 API Key:**
1. 访问 [Google AI Studio](https://makersuite.google.com/app/apikey)
2. 登录 Google 账号
3. 点击 "Create API Key"
4. 复制密钥并粘贴到 `.env` 文件

**注意:** 如果不配置 API Key，系统会使用内置的模拟回复，功能同样可用。

### 3. 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

项目将在 `http://localhost:3000` 启动

### 4. 构建生产版本

\`\`\`bash
npm run build
\`\`\`

## 🎮 使用指南

### 角色介绍

**亚瑟 - 傲娇毒舌王者**
- 性格: 傲慢、毒舌、讽刺
- 特点: 会吐槽玩家，但偶尔给出中肯建议
- 风格: 说话直接不留情面，犀利讽刺

### 对话技巧

1. **触发好评**: 说"好"、"赢"、"棒" → 亚瑟会傲娇认可
2. **触发差评**: 说"坑"、"输"、"菜" → 亚瑟会毒舌吐槽
3. **彩蛋触发**: 输入"抽" → 出现鞭痕特效
4. **终极彩蛋**: 连续"抽"10次 → 亚瑟爆炸！

### 情绪状态

- 😐 **中立** - 默认状态
- 😏 **讽刺** - 吐槽模式
- 😎 **傲娇** - 略有赞许
- 😤 **生气** - 被惹恼了
- 💥 **爆炸** - 忍无可忍

## 🐛 故障排除

**Q: 为什么 AI 回复都是固定的？**  
A: 检查 `.env` 文件中的 API Key 是否正确配置。如果未配置，系统会使用模拟回复。

**Q: 如何获取免费的 Gemini API Key？**  
A: 访问 [Google AI Studio](https://makersuite.google.com/app/apikey)，登录后即可免费获取。

**Q: 抽了10次但没爆炸？**  
A: 确保输入的是中文"抽"字，每次都会计数，达到10次会触发爆炸动画。
