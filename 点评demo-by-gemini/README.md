# GameEcho - 互动游戏评论平台

一个现代化的互动游戏评论平台，将传统评论转化为引人入胜的AI驱动对话。

## ✨ 功能特色

- **暗黑游戏美学**：赛博朋克风格UI，配有霓虹光效和玻璃态效果
- **AI角色互动**：每款游戏都有独特的AI角色回复用户评论
- **富媒体评论**：支持文字、图片和视频内容
- **流畅动画**：由Framer Motion驱动的流畅过渡和交互
- **完全响应式**：针对桌面和移动设备优化

## 🛠️ 技术栈

- **React 18** + TypeScript
- **Vite** - 超快的构建工具
- **Tailwind CSS** - 实用优先的样式框架
- **Framer Motion** - 动画库
- **Lucide React** - 图标库

## 📦 安装

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建
npm run build

# 预览生产构建
npm run preview
```

## 🎮 模拟数据

项目包含预填充的数据：

- **4家游戏公司**：任天堂、暴雪娱乐、米哈游、Capcom
- **4款游戏**：每家公司1-2款游戏
- **20+条评论**：每款游戏有5-6条带AI回复的示例评论

## 🎨 设计特色

- **霓虹光效**：带公司专属主题色的悬停动画
- **玻璃态**：半透明卡片配背景模糊
- **动态网格背景**：微妙的动画背景图案
- **自定义AI回复**：模拟AI思考和个性化回复
- **流畅页面过渡**：优雅的进入/退出动画

## 📱 页面

1. **首页**：网格展示游戏公司，带悬停效果
2. **游戏弹窗**：显示所选公司的游戏
3. **游戏详情**：带AI角色的完整互动评论体验

## 🚀 使用方法

1. 启动开发服务器
2. 在首页浏览游戏公司
3. 点击公司查看其游戏
4. 选择游戏进入互动评论页面
5. 与AI角色聊天并浏览社区评论

## 📝 项目结构

```
src/
├── components/
│   ├── LandingPage.tsx      # 主页
│   ├── CompanyCard.tsx      # 公司展示卡片

│   ├── GameDetail.tsx       # 游戏详情页

├── App.tsx                  # 主应用组件
├── main.tsx                 # 入口点

```

## 🎯 核心交互

- **公司卡片悬停**：光晕效果和轻微缩放动画
- **游戏选择**：背景模糊的弹窗和弹簧动画
- **评论提交**：AI思考指示器，然后是个性化回复
- **评论卡片**：悬停缩放效果和AI回复区域
- **筛选标签**：基于游戏主题的流畅颜色过渡
