# 🚀 快速开始指南

## 前置要求

确保已安装Node.js（版本16或更高）。

## 安装和运行

```bash
# 进入项目目录
cd "/Users/allenzqwei/Desktop/playtest/点评demo-by-gemini"

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

应用将在 `http://localhost:5173` 可用

## 🎮 体验内容

### 首页
1. **鼠标悬停在公司卡片上**查看霓虹光效
2. **点击任意公司**（任天堂、暴雪娱乐、米哈游、Capcom）打开游戏弹窗
3. **选择游戏**进入互动评论体验

### 游戏详情页
1. **与AI角色见面** - 每款游戏都有独特的角色和自定义问候语
2. **写评论** - 使用左下方的富媒体输入栏
3. **观看AI回复** - 提交后，看AI"思考"并回复
4. **浏览评论** - 滚动查看带AI回复的社区评论
5. **筛选评论** - 使用标签按全部/文字/图片/视频筛选

## 🎨 值得注意的功能

- 所有卡片上的**玻璃态效果**
- 匹配每款游戏主题色的**霓虹光晕动画**
- 由Framer Motion驱动的**流畅页面过渡**
- 带动画点的**AI输入指示器**
- **响应式设计** - 试试调整浏览器大小！

## 📊 包含的模拟数据

- 4家游戏公司
- 4款热门游戏（塞尔达王国之泪、超级马力欧兄弟惊奇、原神、崩坏星穹铁道）
- 20+条带AI回复的示例评论
- 富媒体内容（文字、图片、视频标签）

## 🛠️ 技术栈

- React 18 + TypeScript
- Vite（构建工具）
- Tailwind CSS（样式）
- Framer Motion（动画）
- Lucide React（图标）
