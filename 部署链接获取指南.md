   - 公开/私有：任选
   - 点击 "Create repository"

### 步骤 2：推送代码到 GitHub

```bash
cd /Users/allenzqwei/Desktop/playtest/GameSoul游戏之魂

# 初始化 git（如果还没有）
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: GameSoul project"

# 关联 GitHub 仓库（替换为你的用户名）

# 推送代码

```

### 步骤 3：在 Vercel 导入项目

   ```
   https://vercel.com
   ```

4. 选择 `gamesoul-demo` 仓库

5. Vercel 会自动检测配置，点击 "Deploy"

6. 等待 1-2 分钟，获取链接：
   ```
   https://gamesoul-demo.vercel.app
   ```
