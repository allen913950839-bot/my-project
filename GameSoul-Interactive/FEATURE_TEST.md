## 🔍 常见问题诊断

### 问题 1: AI 不回复
**可能原因**:
- API Key 未配置
- API Key 已过期
- API 调用限额已用完

**解决方法**:
1. 检查 Vercel 环境变量
2. 查看控制台错误信息
3. 访问 /api/check-env 检查状态

---

### 问题 2: 保存对话失败
**可能原因**:
- Vercel KV 数据库未配置

**解决方法**:
1. 在 Vercel Dashboard 创建 KV 数据库
2. 等待自动配置环境变量
3. 重新部署项目

---

### 问题 3: 图片不显示
**可能原因**:
- 图片文件不存在
- 图片路径错误

**解决方法**:
1. 检查 `public/` 目录
2. 确认文件名正确
3. 重新上传图片
