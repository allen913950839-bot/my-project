## 🐛 常见问题

### Q1: 页面打不开？
**A:** 确保HTTP服务已启动：
```bash
lsof -i:5173
# 如果没有输出，说明服务未启动，需要重新运行启动命令
```

### Q2: 无法进入游戏详情页？
**A:** 请确保：
1. 点击厂商卡片后，先展开游戏列表
2. 然后点击具体游戏项（如"王者荣耀"）
3. 查看浏览器控制台是否有JavaScript错误

### Q3: 亚瑟AI背景图片不显示？
**A:** 
1. 检查网络连接（图片URL: https://i.imgur.com/pS8gJ3k.jpeg）
2. 如果网络受限，图片可能无法加载
3. 背景色会自动显示为紫色半透明，确保文字可读

### Q4: 如何停止服务？
**A:** 
```bash
# 查找进程ID
lsof -ti:5173

# 停止服务（替换PID为实际进程ID）
kill -9 PID
```
