# 游戏角色背景图片说明

## 🎨 设计概念

每个游戏的详情页都有独特的背景渐变色和典型角色图标，用于强化游戏的视觉识别度和品牌特色。

---

## 🎮 游戏背景配色方案

### 1️⃣ 原神 (Genshin Impact)
- **角色图标**: 🎀 派蒙
- **背景配色**: 紫粉渐变 + 金色光晕
- **设计理念**: 
  - 主色调：紫色→粉色（代表派蒙的蝴蝶结和俏皮个性）
  - 辅助色：金色光晕（象征元素力量和冒险光芒）
  - 视觉效果：梦幻、温暖、充满幻想

### 2️⃣ 王者荣耀 (Honor of Kings)
- **角色图标**: 👑 皇冠
- **背景配色**: 红金渐变 + 蓝色光晕
- **设计理念**:
  - 主色调：红色→金色（象征王者峡谷的荣耀与激情）
  - 辅助色：蓝色光晕（代表魔法和策略）
  - 视觉效果：热血、竞技、霸气

### 3️⃣ 和平精英 (Peacekeeper Elite)
- **角色图标**: 🎯 靶心
- **背景配色**: 绿蓝渐变 + 橙色光晕
- **设计理念**:
  - 主色调：绿色→蓝色（代表战术迷彩和天空）
  - 辅助色：橙色光晕（象征爆炸和战斗激情）
  - 视觉效果：战术、紧张、刺激

### 4️⃣ 崩坏3 (Honkai Impact 3rd)
- **角色图标**: ⚡ 闪电
- **背景配色**: 紫粉渐变（默认样式）
- **设计理念**: 雷电元素、科技感、未来风

### 5️⃣ 英雄联盟 (League of Legends)
- **角色图标**: 🏆 奖杯
- **背景配色**: 紫粉渐变（默认样式）
- **设计理念**: 竞技荣耀、团队协作

### 6️⃣ VALORANT (无畏契约)
- **角色图标**: 🔫 武器
- **背景配色**: 紫粉渐变（默认样式）
- **设计理念**: 战术射击、精准打击

## 💡 技术实现

### CSS渐变背景
```css
/* 王者荣耀背景 */
.banner-honor-of-kings::before {
    background-image: 
        linear-gradient(135deg, rgba(220, 38, 38, 0.8), rgba(234, 179, 8, 0.8)),
        radial-gradient(circle at 20% 30%, rgba(59, 130, 246, 0.4) 0%, transparent 60%);
}
```

### 动态切换逻辑
- 点击不同游戏时，自动切换对应的背景类名
- 角色图标随之变化
- 游戏标题同步更新

### 响应式设计
- 桌面端：角色图标在右侧悬浮（300x300px）
- 移动端：角色图标居中显示（200x200px）

---

## 🎯 用户体验优势

1. **视觉识别**: 每个游戏有独特配色，一眼识别
2. **品牌强化**: 角色图标增强游戏IP认知
3. **沉浸感**: 渐变背景营造游戏氛围
4. **动画效果**: 角色图标有浮动动画，增加生动感
