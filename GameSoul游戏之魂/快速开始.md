# 🚀 GameSoul 快速开始

## 一分钟启动

### 方法 1：使用启动脚本（推荐）

```bash
# 在终端中运行
./启动项目.sh
```

### 方法 2：手动命令

```bash
# 安装依赖（首次运行）
npm install

# 启动项目
npm run dev
```

## 🎮 体验指南

### 步骤 1：选择游戏
1. 看到 "GameSoul" 大标题
2. 点击 "腾讯游戏" 卡片
3. 卡片展开后，点击 "王者荣耀"

### 步骤 2：与妲己对话
1. 进入聊天界面，看到妲己的欢迎语
2. 在底部输入框输入你的游戏体验
3. 尝试输入不同情绪的内容：

**负面示例**：
- "匹配系统太烂了"
- "队友太坑"
- "游戏卡顿"

**正面示例**：
- "这游戏太好玩了"
- "新皮肤超美"
- "体验完美"

4. 观察妲己的表情和回复变化

### 步骤 3：生成记忆卡
1. 发送消息后，点击 "生成灵魂记忆卡"
2. 查看你的专属记忆卡片
3. 可以返回继续探索

## 💡 核心功能

| 功能 | 说明 |
|-----|------|
| 🎯 智能对话 | AI 角色根据你的情绪回复 |

## ❓ 遇到问题？

### Node.js 未安装
访问 https://nodejs.org/ 下载安装

### 端口被占用
修改 `vite.config.js` 中的端口号

### 依赖安装失败
```bash
npm config set registry https://registry.npmmirror.com
npm install
```
