# GameSoul - 游戏之魂 ✨

一个创新的游戏点评平台，通过与游戏角色AI人格对话来分享游戏体验，而不是填写传统的评价表单。

## 🎮 核心概念

GameSoul 重新定义了游戏评价的方式。用户不再面对冰冷的表单，而是与游戏角色的"灵魂"进行真实对话，分享自己的游戏体验。每一次对话都会生成独特的"灵魂记忆卡"，承载玩家的真实感受。

## ✨ 主要功能

### 1. 游戏宇宙选择器
- 视觉震撼的游戏发行商展示网格
- 硬编码演示数据：腾讯游戏 → 王者荣耀
- 优雅的展开/折叠动画效果
- 点击发行商展开游戏列表，选择游戏进入灵魂链接

### 2. 灵魂链接界面（核心体验）
- **氛围背景**：根据游戏主题的渐变色彩（王者荣耀：金色/蓝色）
- **角色展示**：大尺寸动画角色头像（妲己）
- **对话系统**：
  - AI角色主动发起对话："主人，你回来啦！今天峡谷的战斗怎么样？"
  - 支持文本输入和模拟语音输入
  - 智能情感分析（正面/负面/中性）
  - 角色根据用户情绪做出相应反应（表情、回复）
- **妲己人格特征**：
  - 俏皮、妩媚的狐狸精灵
  - 称呼用户为"主人"
  - 使用表情符号增强互动感
  - 根据用户输入展现不同情绪

### 3. 记忆卡片生成
- 点击"生成灵魂记忆"按钮
- 将对话转化为精美的卡片设计
- 包含内容：
  - 角色头像
  - 用户评价引用
  - 星级评分（基于情感分析）
  - 情绪标签
  - 时间戳
- 支持下载和分享功能

## 🛠 技术栈

- **React 18** - 现代化UI框架
- **Vite** - 快速开发构建工具
- **Tailwind CSS** - 实用优先的CSS框架
- **Framer Motion** - 强大的动画库
- **Lucide React** - 精美的图标库

## 🎨 设计理念

- **主题**：赛博朋克 × 奇幻风格
- **配色**：深色背景 + 霓虹色调
- **动画**：流畅的过渡效果（使用 Framer Motion 的 layoutId）
- **体验**：沉浸式、情感化的交互设计
