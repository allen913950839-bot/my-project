## ⚠️ 常见问题

### Q: 页面打不开？
```bash
# 检查服务器是否运行
lsof -i :5173
lsof -i :5174

# 如果没有输出，说明服务器未启动
# 按照"重新启动"部分的指令启动
```

### Q: 端口被占用？
```bash
# 更改端口（以GameVerse为例）
python3 -m http.server 5175

# 访问新端口
http://localhost:5175/gameverse-final.html
```

```bash
# GameSoul（React）会自动热更新

```
