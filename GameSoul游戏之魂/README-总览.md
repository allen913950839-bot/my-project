# 🎮 GameSoul 游戏之魂 - 项目总览

欢迎！这个文件夹包含了两个创新的游戏点评项目。

---

## 🌟 项目介绍

### 1️⃣ GameSoul - 游戏之魂（React 版）
**一句话介绍**：与游戏角色的灵魂对话，分享你的游戏体验

**核心创新**：
- 🦊 与王者荣耀妲己进行灵魂对话
- 💬 AI 根据你的情绪智能回复
- 🎴 生成精美的 3D 记忆卡片
- ✨ 赛博朋克 × 奇幻视觉风格

**技术亮点**：
- React 18 + Vite
- Framer Motion 复杂动画
- Tailwind CSS 主题定制
- 智能情感分析系统

---

### 2️⃣ GameVerse - 游戏点评宇宙（HTML 版）
**一句话介绍**：AI 驱动的多模态游戏点评社区

**核心创新**：
- 🎮 支持多个厂商和游戏（Epic/米哈游/Riot）
- 📝 5种点评类型（文字/图片/视频/语音/GIF）
- 🤖 每个游戏都有专属AI角色
- 🎨 现代玻璃态UI设计

**技术亮点**：
- 纯 HTML/CSS/JS（无依赖）
- 单文件，即开即用
- 完整的产品原型
- 8条完整评论示例
