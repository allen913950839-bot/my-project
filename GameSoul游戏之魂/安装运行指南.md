# GameSoul 安装运行指南 🚀

## 前置要求

在运行本项目之前，需要确保你的系统已安装 Node.js 和 npm。

### 检查是否已安装

打开终端，运行以下命令：

```bash
node --version
npm --version
```

如果显示版本号，说明已安装。如果提示命令不存在，请按照下面的步骤安装。

## 安装 Node.js

### macOS 安装方法

#### 方法 1：使用 Homebrew（推荐）

```bash
# 如果还没安装 Homebrew，先安装它
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 使用 Homebrew 安装 Node.js
brew install node
```

#### 方法 2：从官网下载

访问 [Node.js 官网](https://nodejs.org/) 下载 LTS 版本的安装包，双击安装。

## 安装项目依赖

安装好 Node.js 后，在项目目录运行：

```bash
# 进入项目目录
cd "/Users/allenzqwei/Desktop/playtest/GameSoul游戏之魂"

# 安装依赖
npm install
```

依赖安装需要几分钟时间，请耐心等待。

## 运行项目

### 开发模式

```bash
npm run dev
```

运行后，浏览器会自动打开 `http://localhost:5173`，你就可以看到 GameSoul 应用了！

### 生产构建

```bash
# 构建项目
npm run build

# 预览构建结果
npm run preview
```

## 使用说明

### 1. 选择游戏

- 在首页点击"腾讯游戏"卡片
- 卡片展开后显示"王者荣耀"
- 点击"王者荣耀"进入灵魂链接界面

### 2. 与妲己对话

- 在聊天框输入你的游戏体验
- 妲己会根据你的情绪做出不同反应
- 试试输入正面或负面的评价，看看她的反应！

### 示例对话

**负面评价**：
- "匹配系统太烂了"
- "队友太坑"
- "游戏太卡了"

**正面评价**：
- "这游戏太好玩了"
- "新皮肤超级漂亮"
- "最佳游戏体验"

### 3. 生成记忆卡

- 发送至少一条消息后
- 点击"生成灵魂记忆卡"按钮
- 查看你的专属记忆卡片

## 常见问题

### Q: 启动后浏览器不自动打开？

A: 手动访问 `http://localhost:5173`

### Q: 端口 5173 被占用？

A: 修改 `vite.config.js` 中的 `port` 配置

### Q: 依赖安装失败？

A: 尝试使用国内镜像源：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Q: 页面显示空白？

A: 打开浏览器控制台（F12）查看错误信息

## 开发建议

- 使用现代浏览器（Chrome、Edge、Firefox 最新版）
- 打开浏览器开发者工具查看动画效果
- 尝试调整窗口大小测试响应式设计

1. Node.js 版本是否 >= 16
2. 依赖是否完整安装
3. 浏览器控制台是否有错误信息
