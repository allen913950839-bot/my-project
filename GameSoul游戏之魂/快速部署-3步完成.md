## 🎯 方式二：通过 GitHub 部署（推荐，支持自动更新）

### 准备工作（一次性）

#### 1. 创建 GitHub 仓库
访问：https://github.com/new
- Repository name: `gamesoul-demo`
- 选择 Public
- 点击 "Create repository"

#### 2. 上传代码
在终端执行以下命令：

```bash
cd /Users/allenzqwei/Desktop/playtest/GameSoul游戏之魂

# 初始化 git（如果还没有）
git init

# 添加所有文件
git add .

# 提交
git commit -m "GameSoul 游戏之魂 - 初始版本"

# 关联远程仓库（替换成你的 GitHub 用户名）
git remote add origin https://github.com/你的用户名/gamesoul-demo.git

# 推送代码
git branch -M main
git push -u origin main
```

#### 3. 在 Vercel 导入
1. 访问：https://vercel.com/new
2. 点击 "Import Git Repository"
3. 选择 `gamesoul-demo` 仓库
4. 点击 "Deploy"

**完成！** 以后每次更新代码，只需：
```bash
git add .
git commit -m "更新说明"
git push
```
Vercel 会自动重新部署！

## 🎯 方式三：本地命令行（开发者推荐）

### 快速部署命令

```bash
# 进入项目目录
cd /Users/allenzqwei/Desktop/playtest/GameSoul游戏之魂

# 登录 Vercel（首次需要）
npx vercel login
# 选择登录方式，推荐 GitHub

# 部署到生产环境
npx vercel --prod
```

按提示操作：
- `Set up and deploy?` → **Yes**
- `Which scope?` → 选择你的账户
- `Link to existing project?` → **No**
- `What's your project's name?` → **gamesoul-demo**
- `In which directory is your code located?` → **./（直接回车）**
- `Want to override the settings?` → **No**

等待 30-60 秒，部署完成！

## 🎨 自定义域名（可选）

如果您有自己的域名，可以绑定：

3. 按提示添加 DNS 记录
4. 等待验证完成
